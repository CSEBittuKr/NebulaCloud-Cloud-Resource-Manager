## 📌 Project Overview

NebulaCloud is a web-based cloud resource management simulation system.  
It allows users to:

- Create Virtual Machines
- Allocate Storage
- View VM Details in a dynamic table
- Generate Billing Summary
- Manage cloud-like infrastructure resources

This project simulates how real cloud platforms like AWS manage infrastructure resources.

---

## 🚀 Features

- 🔐 User Login System
- 🖥 Virtual Machine Provisioning
- 💾 Storage Allocation
- 📊 Dynamic Billing Calculation
- 🎨 Modern Glass UI with Gradient Design
- 📋 VM Management Table
- ❌ Delete VM Option
